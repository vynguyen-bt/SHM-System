1. To start the backend server:
   
    > cd backend

    > python app.py

2. To start frontend:
   
    > cd public

    > python -m http.server 3000

    The main frontend will run at http://localhost:3000/

3. For first time user, this is how to install all modules:
   
    > npm install

4. To install library for backend:

    > pip install -r requirements.txt
